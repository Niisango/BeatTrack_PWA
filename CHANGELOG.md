BeatTrack Manager 更新履歴

このプロジェクトの主要なアップデート情報を記録しています。

[v1.2.1] - 2024-05-25

変更

動的更新履歴の実装: HTML内のハードコードを廃止し、この CHANGELOG.md を直接読み込んで表示するように変更。

[v1.2.0] - 2024-05-24

追加

マルチテーマ対応: ライトモードとダークモードの切り替え機能を追加。

テーマ永続化: ユーザーのテーマ選択をブラウザの localStorage に保存する機能。

変更

UI全体のカラーパレットを Tailwind CSS の dark モードに対応。

ヘッダーにテーマ切り替え用のサン/ムーンアイコンを追加。

[v1.1.0] - 2024-05-20

追加

Version画面: アプリのバージョン情報、クレジット、および更新履歴を表示するセクションを追加。

Gemini クレジット: 「このツールはGeminiによって生成されました」という文章を正式に記載。

[v1.0.0] - 2024-05-15

追加

PWA基盤: オフライン対応およびインストールバナーの実装。

データ管理: LocalStorage を利用した楽曲データの CRUD 操作。

インポート/エクスポート: データベース全体を JSON 形式で入出力する機能。

レスポンシブデザイン: モバイル対応グラスモーフィズム UI。

Note: このツールは Gemini によって生成されました。
